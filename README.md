Аддон для виклику різних маунтів в залежності від локації однією кнопкою.
- підтримує флай-форму друля
- працює на Wintergrasp (йде бій - граунд, не йде бій - флай)

Користувач налаштовує 2 списки (флай і граунд) маунтів, виносить макрос на панель. 
В макрос вставляється рандомний маунт (флай/граунд відповідно до флай/граунд зони). 

Макрос оновлюється (вибирається новий маунт):
- при зміні локації
- при переході в даларані з флай/граунд зони
- після комбату
- при виході з приміщень
